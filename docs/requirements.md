# DB Insight - 数据库连接与结构查看工具

## 1. 项目概述

**项目名称**: DB Insight
**项目类型**: Web 应用
**核心功能**: 通过网页连接数据库，可视化查看数据库表结构、列信息及注释
**目标用户**: 数据库开发者、DBA、后端工程师

---

## 2. 功能需求

### 2.1 数据库连接管理

| 功能 | 描述 |
|------|------|
| 支持数据库类型 | MySQL、PostgreSQL |
| 连接方式 | 输入 host、port、username、password、database |
| 连接测试 | 保存前验证连接是否成功 |
| 连接保存 | 支持保存多个连接配置，方便切换 |
| 断开连接 | 一键断开当前连接 |

### 2.2 数据库导航

| 功能 | 描述 |
|------|------|
| 表列表 | 展示当前数据库所有表 |
| 表搜索 | 支持按表名搜索过滤 |
| 列搜索 | 支持跨表搜索列名，输入列名关键词即可过滤 |
| 表数量统计 | 显示当前数据库的表总数 |

### 2.3 表画布视图

| 功能 | 描述 |
|------|------|
| 可视化画布 | 在大画布上展示所有表的结构关系图 |
| 表卡片 | 每个表以卡片形式呈现，显示表名、主键、列概览 |
| 缩放平移 | 支持鼠标滚轮缩放、拖拽平移画布 |
| 布局方式 | 自动排列表卡片位置，避免重叠 |
| 表关联高亮 | 鼠标悬停表卡片时，高亮显示关联的表 |

### 2.4 表结构查看

| 功能 | 描述 |
|------|------|
| 列信息展示 | 列出表的所有列，包括：列名、数据类型、是否主键、是否可为空、默认值 |
| 注释展示 | 显示列的 comment 信息 |
| 表注释 | 显示表的 comment（如果有） |
| 索引信息 | 显示表上的索引 |
| 快速修改表注释 | 点击表注释即可编辑，支持直接保存更新 |

### 2.5 表注释修改

| 功能 | 描述 |
|------|------|
| 行内编辑 | 点击表注释后直接进入编辑模式 |
| 即时保存 | 编辑完成后即时保存，无需额外确认操作 |
| 取消修改 | 支持 ESC 键取消未保存的修改 |
| 修改反馈 | 保存成功/失败后显示明确的状态提示 |

---

## 3. 非功能需求

| 需求 | 描述 |
|------|------|
| 安全性 | 密码加密存储，不明文显示 |
| 安全性 | 防止 SQL 注入攻击，所有用户输入参数使用参数化查询 |
| 响应速度 | 表结构加载 < 1秒 |
| 兼容性 | 现代浏览器（Chrome、Firefox、Safari、Edge） |

---

## 4. 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端 | React 18 + TypeScript + Vite |
| 后端 | Java 21 + Spring Boot 3 |
| 数据库驱动 | mysql-connector-java、postgresql-jdbc |
| 状态管理 | Zustand |
| 画布库 | React Flow + Dagre |

---

## 5. 页面结构

```
┌─────────────────────────────────────────────────────────┐
│  Header: Logo + 连接状态 + 视图切换 + 断开按钮            │
├──────────────┬──────────────────────────────────────────┤
│              │                                          │
│  侧边栏      │  主内容区                                  │
│  - 连接表单   │  - 表列表视图 / 表画布视图 / 表结构详情    │
│  - 表搜索框   │                                          │
│  - 列搜索框   │                                          │
│  - 表清单     │                                          │
│              │                                          │
└──────────────┴──────────────────────────────────────────┘
```

**视图模式**:
- 表列表视图：表格形式展示所有表
- 表画布视图：大画布上可视化展示所有表
- 表详情视图：单个表的完整结构信息

- [ ] 能够成功连接 MySQL/PostgreSQL 数据库
- [ ] 能够显示数据库中的所有表
- [ ] 点击表名后显示完整的列信息
- [ ] 能够正确显示列的 comment
- [ ] 能够快速修改表注释并保存
- [ ] 所有数据库操作使用参数化查询，防止 SQL 注入
- [ ] 支持跨表列名搜索
- [ ] 表画布视图能展示所有表，支持缩放和平移
- [ ] 界面简洁，操作流畅
